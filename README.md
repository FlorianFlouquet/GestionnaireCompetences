# Projet Compétences

## Présentation
Ceci est un présentation

## Prérequis
- docker
- Java 11+
- docker-compose
- un IDE (IntelliJ, Eclipse)

## Installation

### Base de données

Pour lancer la base de données, il faut se placer dans le dossier `docker` et la lancer la commande suivante:
```bash
docker compose up -d
```
